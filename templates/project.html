Temperature Measurement System
We created a temperature sensing system that displays the temperature in degrees Celsius on a hardware display and communicates with a computer that logs the temperature data and shows a graph of the temperature over time. The system sends an alert text message if the temperature goes outside set thresholds. We used an Arduino Uno, LCD display, DHT11 temperature sensor, and computer connected via Bluetooth serial interface with the computer application in Python.
Infrared Sensor/Receiver
For this lab we implemented an “electric eye” light beam-interrupter safety system transmitting an infrared signal from a given transmitter to a designed receiver. We used discrete component filters to implement sufficient filtering so as to react only to the presence or absence of the IR signal and a microcontroller interfaced to a Python process to send an alert text message when the beam is interrupted.
